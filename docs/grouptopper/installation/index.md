## Installation

When you download the addon from the marketplace, you will receive a ZIP file containing two files:
```
GroupTopper-X.X.X.jar
GroupTopper-X.X.X-universal.jar
```
(where `X.X.X` is the version)

These two files are for separated usage:
- `GroupTopper-X.X.X.jar`
  + is designed specifically for **Topper** and its addons. 
  + This is a small addon that is tightly integrated to **Topper**, so it is required to make this file work.
- `GroupTopper-X.X.X-universal.jar`
  + is a standalone addon for other leaderboard plugins that are not **Topper** (e.g. **ajLeaderboards**, **XLeaderboards**). 
  + Only use this file if your primary leaderboard plugin is not **Topper**.

Although these 2 files don't conflict with each other, it's **recommended** to **install only one of these files** depending on your usage.
