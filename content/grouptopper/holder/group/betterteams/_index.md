> [!NOTE]
> Requires [BetterTeams](https://www.spigotmc.org/resources/17129/)

> [!TIP]
> It's recommended to enable `singleOwner` in the BetterTeams configuration to avoid unexpected behavior when checking for group ownership.

## Format

```yaml
type: betterteams
```

## Example

```yaml
holders:
  jump_group:
    top: jump

    type: betterteams
```
