title = "BentoBox"
+++

> [!NOTE]
> Requires [BentoBox](https://bentobox.world/) and one or more Gamemode addons

## Format

```yaml
type: bentobox
gamemode: <gamemode> # the name of the gamemode
```

## Example

```yaml
holders:
  jump_group:
    top: jump

    type: bentobox
    gamemode: bskyblock
```
