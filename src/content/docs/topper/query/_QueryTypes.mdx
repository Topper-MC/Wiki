### `top_name`

- **Format**:
  - <QueryCode p={props} type="global"><QueryVar client:load field="holder" />;top_name;<QueryVar client:load field="position" /></QueryCode>
- **Description**: The name of the player at the specified position.
- **Argument**:
  - `<holder>`: The name of the Holder
  - `<position>`: The position of the player in the Holder
- **Example**:
  - <QueryCode p={props} type="global"><QueryVar client:load field="holder" />;top_name;<QueryVar client:load field="position" /></QueryCode>
    <ul>
      <li>Retrieve the name of the player at position <QueryVar client:load field="position" /> in the Holder named <code><QueryVar client:load field="holder" /></code></li>
    </ul>

### `top_value`

- **Format**:
  - <QueryCode p={props} type="global">&lt;holder&gt;;top_value;&lt;position&gt;</QueryCode>
  - <QueryCode p={props} type="global">&lt;holder&gt;;top_value;&lt;position&gt;;&lt;format&gt;</QueryCode>
- **Description**: The value of the player at the specified position.
- **Argument**:
  - `<holder>`: The name of the Holder
  - `<position>`: The position of the player in the Holder
  - `<format>`: A number format pattern, see [Value Display](/topper/extra/value_display/) for more details on how to apply a format
- **Example**:
  - <QueryCode p={props} type="global"><QueryVar client:load field="holder" />;top_value;<QueryVar client:load field="position" /></QueryCode>
    - Retrieve the value of the player at position <QueryVar client:load field="position" /> in the Holder named <code><QueryVar client:load field="holder" /></code>
  - <QueryCode p={props} type="global"><QueryVar client:load field="holder" />;top_value;<QueryVar client:load field="position" />;#,###</QueryCode>
    - Retrieve the formatted value of the player at position <QueryVar client:load field="position" /> in the Holder named <code><QueryVar client:load field="holder" /></code>, in the format of <code>#,###</code>
